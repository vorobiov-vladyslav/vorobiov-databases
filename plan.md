## 🏗️ ПР4 — DDL та DML (≈ 1.5–2 год)

**Мета:** Створити структуру `publishing` + наповнити даними.

**Кроки:**
1. `CREATE DATABASE publishing; USE publishing;` (готовий код у зошиті, с. 12-13).
2. Створити 8 таблиць CREATE TABLE по черзі: `Authors, Employees, Books, Orders, Contracts, AuthorBook, EmployeeBook, OrderItem`. Код є.
3. Зробити EER діаграму через Workbench: Database → Reverse Engineer → publishing. Зберегти `.mwb`.
4. Виконати INSERT (10 рядків × 8 таблиць) — код готовий на с. 17-22.
5. Виконати `START TRANSACTION; ... COMMIT;` блоки.
6. Запустити перевірочні `SELECT COUNT(*)` та `SELECT *` по кожній таблиці.
7. Скріни: EER діаграма + COUNT-перевірка + кілька SELECT-результатів.
8. Виконати **UPDATE** і **DELETE** приклади (якщо є — додати власні), скріни до/після.

**Результат:** Працююча БД `publishing` з даними + скріни всіх ключових операцій.
