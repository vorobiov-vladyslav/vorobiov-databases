# Бази даних — практичні роботи

Курс GoIT, ІПЗ 4 семестр. Прогрес виконання 7 практичних робіт з тем 3-9.

## Структура

```
.
├── CLAUDE.md           # робочі правила для Claude (середовище, СУБД, формат)
├── plan.md             # план виконання всіх ПР на вихідні
├── publishing.mwb      # MySQL Workbench модель (EER)
├── publishing.png      # експортована ER-діаграма (оригінал)
├── sql/                # усі SQL-файли по практичних
├── screens/            # скріни для зошита
└── notebook/
    ├── zoshyt.docx     # робочий зошит (поточний стан)
    ├── zoshyt.txt      # текстова копія для diff/grep
    └── *.py            # скрипти заповнення docx
```

## Середовище

- **СУБД:** MariaDB 12.2 (drop-in replacement для MySQL)
- **GUI:** MySQL Workbench
- **БД:** `publishing`
- **Підключення:** `mariadb -u vlad -pvlad`
